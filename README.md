# Steps to initiate project for development:

1. npm install
2. create JSON file of affiliate data in Hasura with the following fields:

   -affiliate_id
   -master
   -master_id
   -master_name
   -unit_id
   -unit_name
   -state
   -local
   -council
   -subunit
   -contact
   -number_of_members
   -agreement_eff_date
   -agreement_exp_date
   -report_individually

recommending use of 2022 data

3. Place the JSON file in entry folder of src folder

4. run 'npm run dev'
